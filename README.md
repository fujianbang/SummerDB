# TaurusDB

[![Rust](https://github.com/fujianbang/TaurusDB/actions/workflows/rust.yml/badge.svg)](https://github.com/fujianbang/TaurusDB/actions/workflows/rust.yml)

A distributed SQL database in Rust.

# Architecture

## front-end
* tokenizer
* parser
* code generator

## back-end
* virtual machine
* b-tree
* pager
* os interface
